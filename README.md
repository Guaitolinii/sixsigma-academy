# 🎓 Six Sigma Academy - Master Black Belt Certification Platform

Bem-vindo à **Six Sigma Academy**, uma plataforma educacional de ponta, projetada para preparar profissionais para a certificação **Master Black Belt (MBB)**. 

A aplicação foi construída visando oferecer uma experiência premium, imersiva e responsiva (Mobile, Tablet e Desktop), gamificando o aprendizado com elementos de XP e testes práticos cronometrados de alto rigor estatístico.

---

## 🚀 Funcionalidades Principais

- 📚 **Trilha de Módulos Completa (1 ao 12):** Conteúdo denso e progressivo abordando desde introdução ao Lean Six Sigma, fases do DMAIC, até Design for Six Sigma (DFSS) e Integração com Tecnologia (Indústria 4.0, Big Data).
- 🏆 **Simulado de Certificação (Timer de 4h):** Um ambiente de prova real com 150 questões de alta dificuldade (misturando cálculos de Cpk, DPMO, Amostragem e testes de hipóteses). Correção automática com gabarito.
- ⚡ **Quiz Arena:** Uma experiência estilo Duolingo/Anki para praticar conhecimentos específicos, com animações fluidas, ganho de XP (Experience Points) e níveis progressivos.
- 📐 **Central de Fórmulas e Tabelas:** Uma enciclopédia rápida com mais de 26 fórmulas essenciais, tabelas estatísticas (Z, Fatores de Cartas de Controle) e *Exemplos Práticos Numéricos* interativos.
- 🧠 **Fluxograma de Decisão Estatística:** Ferramenta visual interativa para auxiliar alunos a escolherem o teste de hipótese correto (T-test, ANOVA, Regressão, etc.) baseado nos dados.
- 📖 **Glossário Interativo:** Mais de 130 termos técnicos organizados em ordem alfabética e com barra de busca inteligente.
- 📊 **Tracker de Progresso:** Dashboard que salva automaticamente seu XP, progresso de leitura, e acertos em formulários localmente.
- 💼 **Business Cases (CRM/Projetos):** Simulador de gestão de portfólio de projetos Six Sigma com ferramentas de priorização e cálculo de ROI.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas do ecossistema front-end moderno:

- **React 19**
- **Vite** (Build Tool extremamente rápida)
- **Tailwind CSS v4** (Estilização utility-first)
- **Framer Motion** (Animações fluidas e micro-interações UX/UI)
- **Lucide React** (Ícones premium SVG)
- **Recharts** (Visualização gráfica de dados)
- **React Router DOM v7** (Navegação client-side)

## 📦 Como Rodar o Projeto Localmente

1. Clone o repositório:
```bash
git clone https://github.com/Guaitolinii/sixsigma-academy.git
```

2. Entre no diretório da aplicação:
```bash
cd sixsigma-academy/sixsigma-app
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acesse no seu navegador: `http://localhost:5173`

## 🎨 Princípios de Design (UI/UX)
- **Dark Mode Premium:** Fundo escuro (`#0D0D18`) com acentos em dourado (`#C9A84C`) para transmitir a qualidade e o prestígio de um curso nível "Master Black Belt".
- **Glassmorphism & Bordas Suaves:** Componentes de interface modernos imitando vidro escuro e painéis com elevações sutis para maior engajamento visual.
- **Responsividade Pura:** Código estruturado para funcionar perfeitamente em telas de celulares, tablets e displays Ultra-Wide, mantendo a experiência em qualquer dispositivo.
